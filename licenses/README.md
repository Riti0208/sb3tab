# Bundled license texts

This directory contains the full text of licenses for bundled assets where
the license is **not** the same as sb3tab's own LGPL-3.0. Attribution for
each third-party component is summarized in
[../THIRD_PARTY_LICENSES.md](../THIRD_PARTY_LICENSES.md); this directory
exists so that the canonical license text travels with the binary as
required by some licenses (e.g. OFL §3).

| File | Applies to |
|---|---|
| [`OFL-1.1.txt`](./OFL-1.1.txt) | NotoSansJP-Medium-subset, NotoSans-Medium, NotoSerif-Regular, Griffy-Regular, Handlee-Regular, Knewave-Regular |
| Ubuntu Font License 1.0 (see https://ubuntu.com/legal/font-licence) | Ubuntu-Bold |

For other dependencies, the license text travels with the source code
inside `components/scratch_core/`, `managed_components/`, or the upstream
header file itself. See `THIRD_PARTY_LICENSES.md` for the per-component
inventory.
